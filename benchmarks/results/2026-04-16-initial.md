# Viseer Benchmark Results (2026-04-16)

This is the first lightweight benchmark pass for Viseer.
It uses automatic checks over a small set of real-world tasks and should be read as a trust and behavior snapshot, not a leaderboard claim.

## Run Summary

- Tasks: 8
- Pass: 3
- Mixed: 5
- Fail: 0
- Average automatic score: 0.78
- Median latency: 6709 ms
- SearXNG URL: http://localhost:8080

## First Takeaways

- Viseer looked strongest on freshness-sensitive lookups and on the conservative pricing path: it would rather return `insufficient` than overstate confidence.
- Official-source coverage is still inconsistent for general search and comparison tasks. Several runs produced relevant answers but not the vendor or first-party domains we would ideally want.
- The no-browser extraction path worked well on the Spree demo storefront and only partially on the Saleor demo storefront, which is exactly the kind of limitation this benchmark should keep visible.
